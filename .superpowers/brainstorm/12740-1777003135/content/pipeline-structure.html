Bagaimana pipeline AI ini diorganisir?

Kamu describe 5 tahap: Ide → Design/Video → Script/Caption → Publish → Analisis. Pertanyaannya: semua dalam satu flow, atau tab terpisah?

A

Pipeline Linear — 1 flow, 5 langkah

User bergerak dari kiri ke kanan: Ide → Produksi → Caption → Publish → Analisis. Satu halaman besar dengan stepper. Cocok kalau setiap konten dibuat dari awal sampai akhir dalam satu sesi.

Ide → Produksi → Caption → Publish → Analisis
B

Tab Modular — tiap tahap tab sendiri

Setiap tahap berdiri sendiri di sidebar AI Studio. User bisa masuk ke tab mana saja kapan saja, tidak harus urut. Lebih fleksibel — bisa langsung buka Caption Builder tanpa harus mulai dari Ide.

✦ Pencari Ide
✦ Produksi (Design / Video)
✦ Content Builder (Script + Caption)
✦ Publish Manager
✦ Analisis Konten
C

Hub + Spoke — Dashboard AI sebagai pusat

Ada satu halaman "AI Studio Dashboard" yang jadi pusat. Dari sini user bisa mulai pipeline baru, lihat status semua konten in-progress, atau langsung masuk ke tool tertentu. Paling powerful tapi paling kompleks.
