Desain Final — AI Studio Hub + Brief Generator

Sub-project 1 dari 4. Approve untuk lanjut ke implementation plan.

1 — AI Studio Hub (Kanban)
📍 Route: /ai — menggantikan halaman AI Studio saat ini
📋 Kolom Kanban: Ide → Brief → Caption → Selesai
🃏 Tiap card: judul, entity (BPI/BSI), platform, tombol aksi sesuai stage
➕ Tombol "Konten Baru" — buat card dari awal atau dari Pencari Ide
🔗 Card di kolom Caption → tombol langsung buka Content Builder
2 — Brief Generator (klik "Generate Brief" di card)
📐 Output Design Brief
✓ Gambar AI langsung (DALL-E 3)
✓ Midjourney prompt — tombol copy
✓ Tipografi detail (font, size, hierarchy)
✓ Color palette + hex codes
✓ Layout & komposisi guide
✓ Format & dimensi (Feed, Story, dll)
🎬 Output Video Brief
✓ Script shot-by-shot + timecode
✓ Storyboard visual per scene (DALL-E 3)
✓ Talking points per scene
✓ Editing style guide (CapCut/Premiere)
✓ Durasi & format rekaman
🚀 Setelah brief di-generate
• Brief Design otomatis push ke halaman Reinaldi
• Brief Video otomatis push ke halaman Faizal
• Card pindah ke kolom "Brief" di Kanban
• Notifikasi muncul di halaman tim yang bersangkutan
3 — Update halaman Tim Produksi
/bpi-reinaldi — tambah tab Brief Inbox: list brief masuk, preview gambar AI, status (Pending / In Progress / Done), tombol "Tandai Selesai"
/bpi-faizal — tambah tab Brief Inbox: list brief masuk, script + storyboard per scene, status tracking, tombol "Tandai Selesai"
Saat tim tandai selesai → card di Kanban otomatis naik ke kolom Caption
